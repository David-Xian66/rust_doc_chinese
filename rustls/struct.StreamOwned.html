结构体 StreamOwned 

Source 
pub struct StreamOwned<C: Sized, T: Read + Write + Sized> {
    pub conn: C,
    pub sock: T,
}
展开描述

This type implements io::Read 并 io::Write, encapsulating 并 owning 一个 Connection C 并 an underlying transport T, such as 一个 socket.

Relies on ConnectionCommon::complete_io() 到 perform the necessary I/O.

This allows you 到 use 一个 rustls Connection like 一个 normal stream.
