Module pem 

Source
展开描述

底层 PEM 解码 API。

这些 API 允许以迭代器方式解码 PEM 格式， 这意味着你可以单次扫描文件从中加载多种不同类型的 PEM section。

结构体§

ReadIter
通过读取 rd 提取并返回所有 PEM section。
SliceIter
对 &[u8] 切片中所有 PEM section 进行迭代的迭代器。

枚举§

Error
解析 PEM 文件内容时可能产生的错误
SectionKind
PEM 文件中一个被识别的 section。

特性§

PemObject
可以从 PEM 数据解码出的条目。

from_buf
