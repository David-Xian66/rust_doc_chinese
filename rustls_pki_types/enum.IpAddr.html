枚举 IpAddr 

Source 
pub enum IpAddr {
    V4(Ipv4Addr),
    V6(Ipv6Addr),
}
展开描述

std::net::IpAddr 的 no_std 实现。

注意：由于我们计划在该类型一旦稳定后用 core::net::IpAddr 替换它，因此该类型本身的标识不应被视为 semver 稳定。不过，其对外提供的接口是稳定的，它们是 core::net::IpAddr 所提供接口的一个子集。
