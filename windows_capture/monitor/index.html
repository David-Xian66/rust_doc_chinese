Module monitor 

源码
展开描述

包含用于处理显示器和屏幕信息的功能： crate::monitor::Monitor。 用于查询和操作显示显示器的实用工具。

提供 [Monitor] 用于检索显示器元数据，如友好名称、设备名称、分辨率、刷新率， 以及将显示器转换为捕获项。

常见任务包括：

  • 通过 [Monitor::enumerate] 枚举显示器。
  • 通过基于 1 的索引选择：[Monitor::from_index]。
  • 通过 [Monitor::primary] 获取主显示器。

要获取显示器的 crate::GraphicsCaptureItem，请对 [Monitor] 使用 [crate::settings::TryIntoCaptureItemWithDetails] 的实现。
