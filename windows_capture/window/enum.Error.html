枚举 Error 

源码 
pub enum Error {
    NoActiveWindow,
    NotFound(String),
    FailedToConvertWindowsString,
    WindowsError(Error),
}
展开描述

通过 Window 查询或操作顶级窗口时可能发生的错误。

变体§

§

NoActiveWindow

调用时不存在前台窗口。

由 Window::foreground 返回。

§

NotFound(String)

没有窗口匹配提供的标题或子串。

由 Window::from_name 和 Window::from_contains_name 返回。

§

FailedToConvertWindowsString

将 UTF-16 Windows 字符串转换为 String 失败。

§

WindowsError(Error)

Windows API 调用返回错误。

包装了 windows::core::Error。
