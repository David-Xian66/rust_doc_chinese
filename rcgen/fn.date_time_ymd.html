pub fn date_time_ymd(year: i32, month: u8, day: u8) -> OffsetDateTime
展开描述

用于从年、月、日值构造 OffsetDateTime 的辅助函数

年、月、日取值假定为 UTC。

该辅助函数有两个用途：第一，无需自己导入 time crate 即可指定日期信息；第二，避免用户手动输入不成比例的y long code just to generate an instance of OffsetDateTime.
