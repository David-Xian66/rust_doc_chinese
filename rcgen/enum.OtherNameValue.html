枚举 OtherNameValue 

Source 
#[non_exhaustive]
pub enum OtherNameValue {
    Utf8String(String),
}
展开描述

一个 OtherName 值，定义见 RFC 5280 §4.1.2.4。

虽然标准规定它可以是任意 ASN.1 类型，但 rcgen 将值限定为 UTF-8 编码字符串，因为这能覆盖最常见的用例，例如智能卡的用户主体名（UPN).
