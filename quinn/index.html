QUIC дј иҫ“еҚҸи®®е®һзҺ°

QUIC жҳҜдёҖз§ҚзҺ°д»Јдј иҫ“еҚҸи®®пјҢй’ҲеҜ№ TCP зҡ„дёҚи¶ід№ӢеӨ„иҝӣиЎҢдәҶж”№иҝӣпјҢдҫӢеҰӮйҳҹеӨҙйҳ»еЎһгҖҒе®үе…ЁжҖ§ејұгҖҒжҸЎжүӢж…ўд»ҘеҸҠжӢҘеЎһжҺ§еҲ¶ж•ҲзҺҮдҪҺдёӢзӯүгҖӮжң¬ crate жҸҗдҫӣдәҶдёҖеҘ—еҸҜз§»жӨҚзҡ„з”ЁжҲ·жҖҒе®һзҺ°гҖӮе®ғжһ„е»әдәҺ quinn-proto д№ӢдёҠпјҢеҗҺиҖ…е®һзҺ°дәҶдёҺе…·дҪ“иҝҗиЎҢж—¶ж— е…ізҡ„еҚҸи®®йҖ»иҫ‘гҖӮ

жң¬ crate зҡ„е…ҘеҸЈзӮ№жҳҜ EndpointгҖӮ

В§е…ідәҺ QUIC

QUIC иҝһжҺҘжҳҜдёӨдёӘз«ҜзӮ№д№Ӣй—ҙзҡ„е…іиҒ”гҖӮеҸ‘иө·иҝһжҺҘзҡ„з«ҜзӮ№з§°дёәе®ўжҲ·з«ҜпјҢжҺҘеҸ—иҝһжҺҘзҡ„з«ҜзӮ№з§°дёәжңҚеҠЎеҷЁгҖӮеҚ•дёӘз«ҜзӮ№еҸҜд»Ҙй’ҲеҜ№дёҚеҗҢиҝһжҺҘеҗҢж—¶е……еҪ“е®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁпјҢдҫӢеҰӮеңЁеҜ№зӯүеә”з”ЁдёӯгҖӮдёәдәҶдј иҫ“еә”з”Ёж•°жҚ®пјҢжҜҸдёӘз«ҜзӮ№йғҪеҸҜд»Ҙжү“ејҖжөҒпјҢдҪҶжөҒзҡ„ж•°йҮҸеҸ—еҜ№з«ҜйҷҗеҲ¶гҖӮйҖҡеёёиҜҘйҷҗеҲ¶дјҡйҡҸзқҖж—§жөҒзҡ„з»“жқҹиҖҢеўһеҠ гҖӮ

жөҒеҸҜд»ҘжҳҜеҚ•еҗ‘жҲ–еҸҢеҗ‘зҡ„пјҢеҲӣе»әејҖй”ҖеҫҲдҪҺдё”з”Ёе®ҢеҚіеҸҜдёўејғгҖӮдҫӢеҰӮпјҢдј з»ҹйқўеҗ‘ж•°жҚ®жҠҘзҡ„еә”з”ЁеҸҜд»ҘдёәжҜҸжқЎиҰҒеҸ‘йҖҒзҡ„ж¶ҲжҒҜдҪҝз”ЁдёҖдёӘж–°жөҒпјҢиҖҢж— йңҖеҶҚиҖғиҷ‘ MTU й—®йўҳгҖӮеҸҢеҗ‘жөҒзҡ„иЎҢдёәдёҺдј з»ҹ TCP иҝһжҺҘйқһеёёзӣёдјјпјҢйҖӮеҗҲеҸ‘йҖҒйңҖиҰҒз«ӢеҚіе“Қеә”зҡ„ж¶ҲжҒҜпјҢдҫӢеҰӮ HTTP иҜ·жұӮгҖӮжөҒж•°жҚ®жҳҜеҸҜйқ дәӨд»ҳзҡ„пјҢдёҚеҗҢжөҒд№Ӣй—ҙзҡ„ж•°жҚ®дёҚдҝқиҜҒйЎәеәҸгҖӮ

йҖҡиҝҮйҒҝе…ҚйҳҹеӨҙйҳ»еЎһе№¶еңЁеҗҢдёҖиҝһжҺҘзҡ„жүҖжңүжөҒд№Ӣй—ҙжҸҗдҫӣз»ҹдёҖзҡ„жӢҘеЎһжҺ§еҲ¶пјҢQUIC иғҪеӨҹеңЁзӣёеҗҢдёӨдёӘдё»жңәд№Ӣй—ҙзҡ„дёҖжқЎжҲ–еӨҡжқЎ TCP иҝһжҺҘд№ӢдёҠжҸҗдҫӣжӣҙй«ҳзҡ„еҗһеҗҗйҮҸе’ҢжӣҙдҪҺзҡ„е»¶иҝҹпјҢеҗҢж—¶зӣёжҜ”еҺҹе§Ӣ UDP еҘ—жҺҘеӯ—жҸҗдҫӣжӣҙжңүз”Ёзҡ„иҜӯд№үгҖӮ

Quinn иҝҳжҸҗдҫӣдёҚеҸҜйқ зҡ„ж•°жҚ®жҠҘжҺҘеҸЈпјҢиҝҷжҳҜдёҖз§Қеә•еұӮеҺҹиҜӯпјҢйҖӮеҗҲеңЁдёҚйңҖиҰҒеҜ№жҹҗдәӣж•°жҚ®иҝӣиЎҢиҮӘеҠЁеҲҶзүҮе’ҢйҮҚдј ж—¶дҪҝз”ЁгҖӮ

QUIC зӣҙжҺҘеҹәдәҺ TLS 1.3 жһ„е»әеҠ еҜҶдёҺиә«д»ҪйӘҢиҜҒжңәеҲ¶гҖӮжӯЈеҰӮ TLS жңҚеҠЎеҷЁдёҖж ·пјҢQUIC жңҚеҠЎеҷЁжңҖеҘҪдҪҝз”Ёз”ұеҸҜдҝЎжңәжһ„зӯҫеҸ‘зҡ„иҜҒд№ҰжқҘж ҮиҜҶиә«д»ҪгҖӮеҰӮжһңиҝҷдёҖзӮ№йҡҫд»ҘеҒҡеҲ°вҖ”вҖ”дҫӢеҰӮжңҚеҠЎеҷЁжҳҜзҹӯз”ҹе‘Ҫе‘Ёжңҹзҡ„жҲ–дёҚз»‘е®ҡеҹҹеҗҚвҖ”вҖ”йӮЈд№ҲдёҺ TLS дёҖж ·пјҢеҸҜд»ҘдҪҝз”ЁиҮӘзӯҫеҗҚиҜҒд№ҰжқҘеҚ•зәҜжҸҗдҫӣеҠ еҜҶгҖӮ
