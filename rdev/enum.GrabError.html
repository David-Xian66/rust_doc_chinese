pub enum GrabError {
    EventTapError,
    LoopSourceError,
    MissingDisplayError,
    KeyboardError,
    KeyHookError(u32),
    MouseHookError(u32),
    SimulateError,
    IoError(Error),
}
展开描述

尝试抓取操作系统事件时可能发生的错误。注意在 Mac 上，不设置辅助功能不会导致错误，只是会忽略事件。

变体（非穷尽）§

此枚举被标记为非穷尽
非穷尽枚举将来可能会添加更多变体。因此，在对非穷尽枚举的变体进行匹配时，必须添加一个额外的通配符分支以应对未来可能新增的变体。
