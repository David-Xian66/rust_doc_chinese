Source
展开描述

Tokio 对 Windows 命名管道 的支持。

结构体§

ClientOptions
一个构建器，用于从客户端构建命名管道并与之交互。
NamedPipeClient
一个 Windows 命名管道 客户端。
NamedPipeServer
一个 Windows 命名管道 服务端。
PipeInfo
命名管道的相关信息。
ServerOptions
一个构建器结构，用于构造带有命名管道特定选项的命名管道。需要修改管道相关选项的命名管道服务端必须使用它。

枚举§

PipeEnd
指示命名管道的端点。
PipeMode
命名管道的管道模式。
