结构体 WeakSender 

源代码 
pub struct WeakSender<T> { /* 私有字段 */ }
展开描述

不会阻止通道被关闭的 sender。

如果 channel 的所有 Sender 实例都已被 drop，仅剩下 WeakSender 实例，则 channel 关闭。

为了发送消息，WeakSender 需要使用 WeakSender::upgrade 进行升级，该函数返回 Option<Sender>。如果所有 Sender 都已被 drop，则返回 None，否则返回 Sender。

§示例

use tokio::sync::broadcast::channel;

let (tx, _rx) = channel::<i32>(15);
let tx_weak = tx.downgrade();

// 升级会成功，因为 `tx` 仍然存在。
assert!(tx_weak.upgrade().is_some());

// 如果 drop 掉 `tx`，则升级会失败。
drop(tx);
assert!(tx_weak.clone().upgrade().is_none());
