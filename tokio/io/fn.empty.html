函数 empty 

Source 
pub fn empty() -> Empty
展开描述

创建一个读取时始终处于 EOF、写入时忽略所有数据的值。

对返回实例的所有写入都将返回 Poll::Ready(Ok(buf.len()))， 并且不会检查缓冲区的内容。

从返回实例的所有读取都将返回 Poll::Ready(Ok(0))。

这是 std::io::empty 的异步版本。

§示例

一个不太实用的"不向缓冲区读取任何内容"的示例：

use tokio::io::{self, AsyncReadExt};

    let mut buffer = String::new();
    io::empty().read_to_string(&mut buffer).await.unwrap();
    assert!(buffer.is_empty());

一种获取缓冲区长度的迂回方式：

use tokio::io::{self, AsyncWriteExt};

let buffer = vec![1, 2, 3, 5, 8];
let num_bytes = io::empty().write(&buffer).await.unwrap();
assert_eq!(num_bytes, 5);
