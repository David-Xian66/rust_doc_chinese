结构体 Empty 

Source 
pub struct Empty { /* private fields */ }
展开描述

Empty 忽略通过 AsyncWrite 写入的任何数据， 并且在通过 AsyncRead 读取时始终为空 （返回零字节）。

此结构体通常通过调用 empty 创建。有关更多详细信息， 请参阅 empty() 的文档。

这是 std::io::empty 的异步版本。
