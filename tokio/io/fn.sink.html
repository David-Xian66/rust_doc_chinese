pub fn sink() -> Sink
展开描述

创建一个异步 writer 实例，它将成功消费所有数据。

对返回实例的所有 poll_write 调用都将返回 Poll::Ready(Ok(buf.len()))， 并且不会检查缓冲区的内容。

这是 std::io::sink.

§示例

use tokio::io::{self, AsyncWriteExt};

let buffer = vec![1, 2, 3, 5, 8];
let num_bytes = io::sink().write(&buffer).await?;
assert_eq!(num_bytes, 5);
