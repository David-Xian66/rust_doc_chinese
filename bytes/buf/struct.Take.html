结构体 Take 

Source 
pub struct Take<T> { /* private fields */ }
展开描述

一个限制从底层缓冲区读取字节数的 Buf 适配器。

此结构体通常通过对 Buf 调用 take() 创建。详情请参阅 take() 的文档。

impl<T> Take<T>

pub fn into_inner(self) -> T

消耗此 Take，返回底层的值。

§示例
use bytes::{Buf, BufMut};

let mut buf = b"hello world".take(2);
let mut dst = vec![];

dst.put(&mut buf);
assert_eq!(*dst, b"he"[..]);

let mut buf = buf.into_inner();

dst.clear();
dst.put(&mut buf);
assert_eq!(*dst, b"llo world"[..]);
Source

pub fn get_ref(&self) -> &T

获取对底层 Buf 的引用。

不建议直接从底层 Buf 读取。

§示例
use bytes::Buf;

let buf = b"hello world".take(2);

assert_eq!(11, buf.get_ref().remaining());
Source

pub fn get_mut(&mut self) -> &mut T

获取对底层 Buf 的可变引用。

不建议直接从底层 Buf 读取。

§示例
use bytes::{Buf, BufMut};

let mut buf = b"hello world".take(2);
let mut dst = vec![];

buf.get_mut().advance(2);

dst.put(&mut buf);
assert_eq!(*dst, b"ll"[..]);
Source

pub fn limit(&self) -> usize

返回可读取的最大字节数。

§Note

若内部 Buf 的字节数少于本方法所指示的数量，则以实际可用字节数为准。

§示例
use bytes::Buf;

let mut buf = b"hello world".take(2);

assert_eq!(2, buf.limit());
assert_eq!(b'h', buf.get_u8());
assert_eq!(1, buf.limit());
Source

pub fn set_limit(&mut self, lim: usize)

设置可读取的最大字节数。

§Note

若内部 Buf 的字节数少于 lim，则以实际可用字节数为准。

§示例
use bytes::{Buf, BufMut};

let mut buf = b"hello world".take(2);
let mut dst = vec![];

dst.put(&mut buf);
assert_eq!(*dst, b"he"[..]);

dst.clear();

buf.set_limit(3);
dst.put(&mut buf);
assert_eq!(*dst, b"llo"[..]);
